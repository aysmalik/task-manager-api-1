API Reference

Base URL: http://localhost:5000

GET /health
Service health status.
Path params: none
Query params: none
Body: none
200: {"status":"ok"}
curl http://localhost:5000/health
GET /info
App info and runtime config.
Path params: none
Query params: none
Body: none
200: {"app":"Task Manager API","version":"1.0.0","max_tasks":100}
curl http://localhost:5000/info
GET /tasks
List all tasks.
Path params: none
Query params: none
Body: none
200: [{...task}]
curl http://localhost:5000/tasks
GET /tasks/{id}
Get one task by ID.
Path params: id (integer)
Query params: none
Body: none
200: {...task}
404: {"error":"Task not found"}
curl http://localhost:5000/tasks/1
POST /tasks
Create a new task.
Path params: none
Query params: none
Body: {"title":"string","description":"string(optional)"}
201: {...created_task}
400: {"error":"'title' is required"}
409: {"error":"Task limit (N) reached"}
curl -X POST http://localhost:5000/tasks \
  -H "Content-Type: application/json" \
  -d '{"title":"Buy milk","description":"2 litres"}'
PUT /tasks/{id}
Not implemented in v1.0.0 (use PATCH for partial update).
Status: endpoint not available
PATCH /tasks/{id}
Update done/undone state of task.
Path params: id (integer)
Query params: none
Body: {"done":true|false}
200: {...updated_task}
400: {"error":"'done' field is required"}
404: {"error":"Task not found"}
curl -X PATCH http://localhost:5000/tasks/1 \
  -H "Content-Type: application/json" \
  -d '{"done":true}'
DELETE /tasks/{id}
Delete a task by ID.
Path params: id (integer)
Query params: none
Body: none
200: {"message":"Task deleted","task":{...}}
404: {"error":"Task not found"}
curl -X DELETE http://localhost:5000/tasks/1
